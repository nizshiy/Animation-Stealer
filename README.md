# Animation-Stealer
# 📖 How to Use
1. **Execute** the script using the download line below into your Roblox executor (Synapse X, Krnl, Xeno, Fluxus, etc.):
```
loadstring(game:HttpGet('https://raw.githubusercontent.com/nizshiy/Animation-Stealer/refs/heads/main/AnimStealer.lua'))()
```
2. **Enter the username** of the user whose animation you want to copy. <i style="color: rgba(0, 0, 0, 0.5);">(the topmost setting in the menu)</i>
3. **Set the Max Duration** <i style="color: rgba(0, 0, 0, 0.5);">(How long to scan)</i>. and **FPS** <i style="color: rgba(0, 0, 0, 0.5);">(Sampling rate)</i>.
4. Click **START RECORDING.**
5. Click **COPY TO CLIPBOARD.**
6. Open Roblox Studio, open the ***Animation Importer Plugin***, and paste your data.

**🛠 Required Plugin:** https://github.com/nizshiy/Animation-Stealer/blob/main/StealerPlugin

<i style="color: rgba(0, 0, 0, 0.5);">Install the plugin yourself, as a Local Plugin</i>.
